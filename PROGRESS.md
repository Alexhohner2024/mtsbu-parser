## Проблема
Нужно проверять страховку ОСЦПВ по госномеру через `policy.mtsbu.ua`.
Сайт защищён Cloudflare Turnstile (капча), которая блокирует автоматические запросы.

## Решение
Создан локальный скрипт `check_policy.py` на базе Playwright.

### Как работает:
1. Открывает реальный браузер (Chromium)
2. Переходит на `policy.mtsbu.ua`
3. Заполняет госномер и дату
4. Ждёт решения Turnstile (автоматически или вручную)
5. Отправляет форму
6. Парсит результат: статус полиса, страховая компания, данные ТС
7. Выводит в консоль + сохраняет JSON

### Файлы:
| Файл | Описание |
|------|----------|
| `check_policy.py` | Основной скрипт проверки |
| `requirements.txt` | Python зависимости |
| `lib/parseMtsbuHtml.js` | Парсер HTML (из старого проекта) |
| `api/parse.js` | Vercel serverless (из старого проекта) |

### Использование:
```bash
# Установка
pip3 install -r requirements.txt
python3 -m playwright install chromium

# Проверка по госномеру
python3 check_policy.py AA1234BC

# С указанием даты
python3 check_policy.py AA1234BC --date 16.05.2026

# Сохранить результат в JSON
python3 check_policy.py AA1234BC --json result.json

# Режим отладки (браузер с окном)
python3 check_policy.py AA1234BC --headless
```

### Вывод:
- **Консоль:** форматированный текст с эмодзи
- **JSON:** полный результат с метаданными

## Проблемы и ограничения

### Cloudflare Turnstile
- **Не решается полностью автоматически** в автоматизированном браузере
- Для 1-10 запросов в день: скрипт открывает браузер, вы кликаете на капчу, скрипт продолжает
- **Бесплатно**, без сторонних сервисов

### Альтернативы (на будущее):
| Метод | Цена | Надёжность |
|-------|------|------------|
| Capsolver API | ~$1-3/1000 запросов | Высокая |
| Свой сервер + Playwright | ~$5/мес VPS | Высокая |
| Официальный API MTSBU | ? | Лучшая |
